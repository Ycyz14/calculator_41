# 简易计算器_25

一个基于 Web 的简易计算器应用，支持基本数学运算和响应式设计。

## ✨ 功能特性

- ➕ 基本四则运算（加、减、乘、除）
- 🔢 小数点运算
- 🧮 百分比计算
- ⬅️ 退格功能
- 🆑 清空功能
- 📱 响应式设计，支持移动端

- 可选：Node.js（用于本地开发）

# 简易计算器_41

一个基于 Web 的简易计算器应用，支持基本数学运算和响应式设计。

## ✨ 功能特性

- ➕ 基本四则运算（加、减、乘、除）
- 🔢 小数点运算
- 🧮 百分比计算
- ⬅️ 退格功能
- 🆑 清空功能
- ⌨️ 键盘操作支持
- 📱 响应式设计，支持移动端

## 🚀 快速开始

### 环境要求
- 现代浏览器（Chrome、Firefox、Safari等）
- 可选：Node.js（用于本地开发）

### 安装运行
1. 克隆项目：
```bash
git clone https://github.com/Ycyz14/calculator_41.git
```

2. 进入项目目录：
```bash
cd calculator_41
```

3. 打开 `src/index.html` 在浏览器中运行

## 🛠️ 技术栈

- HTML5
- CSS3 (Flexbox布局, Grid布局)
- JavaScript (ES6+)

## 📖 使用说明

1. 点击数字按钮输入数字
2. 点击运算符按钮选择运算
3. 点击等号查看结果
4. 使用 C 按钮清空，CE 按钮退格

**键盘支持：**
- 数字键 `0-9`：输入数字
- 运算符 `+`, `-`, `*`, `/`：选择运算
- `Enter` 或 `=`：计算结果
- `Escape` 或 `C`：清空
- `Backspace`：退格
